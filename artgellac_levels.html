ARTGELLAC LEVELS

De nagelsets bij Artgellac zijn verdeelt in 3 verschillende levels, Artgellac level 1, 2, en 3. Elk level heeft zijn eigen prijs. Welk level kies jij?

ARTGELLAC LEVEL 1

Level 1 bestaat uit een nagel set van 1 kleur. Je kunt kiezen uit veel verschillende kleuren. De set geeft een 'clean look' en kost maar 18 euro!

ARTGELLAC LEVEL 2

Level 2 bestaat uit een basic nail art, waar er gebruik word gemaakt van 2 verschillende kleuren. Voorbeelden van een basic design zijn, vlammetjs, bliksems, french en sworls enz. Deze set kost 20 euro.

ARTGELLAC LEVEL 3

Level 3 bestaat uit een wat moeilijker design, denk aan het aura effect, nailart met meerdere kleuren, chrome nails an andere complexe designs. Deze set heeft een prijs van 25 euro.
